# Tech Stack

## Engine & Language
- Unity 2022.3.62f3c1 (LTS)
- C# 9.0, .NET Standard 2.1
- Target: StandaloneWindows64

## Key Unity Systems
- New Input System (`UnityEngine.InputSystem`) — action map defined in `Assets/Outer Witness.inputactions`
  - Player actions: Move (Vector2), Look (Vector2), Fire (Button), Jump (Button)
  - Some scripts (JetpackController) still use legacy `Input.GetKey` for Shift/Ctrl/Q/E
- Universal Render Pipeline (URP) — materials use `Universal Render Pipeline/Lit`
- TextMeshPro for UI text
- Unity Physics (3D Rigidbody, no Physics2D usage)

## Rendering
- Space skybox from "Free Skyboxes - Space" asset
- HDR emission on the sun via `MaterialPropertyBlock`
- `[ExecuteAlways]` on visual controllers for editor preview (SunController, SolarLightSynchronizer)

## Build & Commands
- No custom build scripts detected; use Unity Editor for building
- The `.csproj` is auto-generated by Unity — do not edit manually
- Editor tools available under `Tools > Outer Witness > Planet Tools`

## Dependencies
- TextMesh Pro (bundled)
- Visual Scripting (com.unity.visualscripting 1.9.4, likely unused)
- Code Coverage (com.unity.testtools.codecoverage 1.2.6)
